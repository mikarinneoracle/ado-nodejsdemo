# Node.js Demo App

A simple Node.js web application using Express.

## Installation

Install dependencies:
```bash
npm install
```

## Running the App

Start the server:
```bash
npm start
```

The app will be available at `http://localhost:3000`

## Project Structure

```
nodejsdemo/
├── server.js          # Express server
├── package.json       # Dependencies and scripts
├── public/
│   └── index.html     # Web page
└── README.md          # This file
```
